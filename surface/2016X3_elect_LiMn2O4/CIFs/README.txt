download packmol
(http://leandro.iqm.unicamp.br/m3g/packmol/download.shtml)
1. Download
2. input your information
3. get packmol.tar.gz or packmol.exe
  Multi-platform source code: packmol.tar.gz (e.g., Linux)
  Windows 10 x86_64 executable: packmol.exe

usage (windows 10)
1. shift + right click
  Power Shell
2. .\run.bat
(you can get "electout.xyz")

compiling (ubuntu 18.04 LTS)
1. tar zxvf packmol.tar.gz
2. cd packmol
3. ./configure
4. make

usage (ubuntu 18.04 LTS)
1. packmol < inputfile.inp
